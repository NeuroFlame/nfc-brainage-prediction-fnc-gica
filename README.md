# Brain Age Prediction FNC GICA

Brain age prediction using decentralized SVR with FNC (functional network connectivity) matrices as features. Ported from [coinstac-brainage-fnc](https://github.com/trendscenter/coinstac-brainage-fnc) to the NeuroFLAME / NVFlare federated learning framework.

---

## Algorithm

A two-round federated computation:

**Round 0** — Non-owner sites each train a local `LinearSVR` on their full dataset and send back the learned weight vector. The owner site caches its train/test split for round 1.

**Round 1 (server)** — The server stacks all local weight vectors into a matrix `w_locals` (shape: `n_features × n_sites`) and broadcasts it to all sites.

**Round 1 (owner site)** — The owner projects its cached data through the averaged local weights (`U = X @ mean(w_locals)`), then trains a second `LinearSVR` on the projected features. Final metrics are written to disk.

## Input Data

Each site directory under `test_data/` must contain:

- `coinstac-gica_postprocess_results.mat` — MATLAB file with `fnc_corrs_all` array (subjects × sessions × components × components)
- `covariates.csv` — CSV with at minimum `filename` and `age` columns

---

## Computation Parameters

Edit `test_data/server/parameters.json` to configure the run:

| Parameter | Description | Default |
|-----------|-------------|---------|
| `owner_site` | Site name of the holdout/owner site | `"site1"` |
| `data_file` | FNC `.mat` filename | `"coinstac-gica_postprocess_results.mat"` |
| `label_file` | Covariates CSV filename | `"covariates.csv"` |
| `input_source` | `"GICA"` or `"UKBioBank_Comp2019"` | `"GICA"` |
| `split_type` | `"random"` or `"age_range_stratified"` | `"random"` |
| `test_size` | Fraction of subjects for test set | `0.1` |
| `shuffle` | Shuffle before splitting | `true` |
| `svr_params_local` | `LinearSVR` kwargs for non-owner sites | see file |
| `svr_params_owner` | `LinearSVR` kwargs for owner site (round 1) | see file |
| `site_id_name_map` | Optional map of NVFlare site IDs to human-readable names | `{}` |

---

## Running Locally (Simulation)

**Step 1:** Generate the job folder for your sites:

```bash
python makeJob.py site1,site2,site3
```

**Step 2:** Run the NVFlare simulator:

```bash
python debug.py job -w simulator_workspace -c site1,site2,site3 -n 3 -t 3
```

Results are written to `test_output/<job_id>/<site_name>/`.

## Notes on COINSTAC → NeuroFLAME Translation

| COINSTAC | NeuroFLAME / NVFlare |
|----------|----------------------|
| `local.py local_0()` | `executor.py` → `TASK_NAME_GET_LOCAL_SVR_WEIGHTS` |
| `local.py local_1()` | `executor.py` → `TASK_NAME_ACCEPT_AGGREGATED_WEIGHTS` |
| `remote.py remote_0()` | `aggregator.py` round-0 `accept()` + `aggregate()` |
| `remote.py remote_1()` | `aggregator.py` round-1 `accept()` |
| `state['owner']` / `clientId` check | `computation_parameters["owner_site"]` vs `fl_ctx.get_identity_name()` |
| `args['cache']` (disk) | `fl_ctx.set_prop("cached_splits", ...)` (in-memory, sticky) |
| stdin/stdout JSON protocol | `Shareable` dict |
| `compspec.json` inputs | `test_data/server/parameters.json` |
